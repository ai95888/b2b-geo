# B2B GEO Skill

B2B生成式引擎优化(GEO)完整工作流，帮助跨境B2B卖家在AI搜索时代获取流量。

## 功能

- 🔍 挖掘海外买家高频采购问题
- ✏️ 产品页改写为AI可直接引用的结构化格式
- 📱 Reddit/Quora等第三方平台权威信号建设

## 使用方法

在OpenClaw中，当需要执行GEO优化工作时，此skill会自动触发。

或手动加载：
```
使用 b2b-geo skill
```

## 适用场景

- B2B跨境卖家想获取AI流量红利
- 询盘减少，搜索引擎SEO效果下降
- 想知道如何被AI推荐到答案前几位
